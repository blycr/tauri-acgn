# 项目概览

## 背景
本项目面向家庭局域网（LAN），构建一个 **ACGN（动画、漫画、游戏、小说）资源管理平台**。目标是统一管理、浏览、播放以及元数据编辑，提供跨平台（Windows、Linux、macOS）桌面客户端，同时支持局域网共享，让家庭成员可以在任意设备上访问资源。

## 核心功能
- **资源库管理**：自动扫描本地指定文件夹，识别并导入视频、漫画图片、小说文本等资源。
- **多媒体播放**：内置视频播放器、漫画阅读器、小说阅读器，支持全屏、暗黑模式、章节跳转等交互。
- **元数据编辑**：手动或通过外部 API（如 MyAnimeList、AniList）补全标题、封面、简介、标签等信息。
- **收藏/历史**：用户可收藏喜欢的作品，系统记录阅读/观看历史，提供继续观看入口。
- **多库支持**：可创建多个资源库，分别对应不同磁盘或网络共享目录。
- **局域网共享**：通过局域网 IP 访问，支持移动端浏览器访问，实现家庭共享。

## 技术栈
- **前端**：HTML、Vanilla CSS（玻璃拟态 + 动态微交互）、JavaScript（ESM），开发阶段使用 Vite 开发服务器。
- **后端**：Rust（Tauri 主进程），SQLite 用于存储元数据、进度、收藏等。
- **跨平台**：Tauri 2.0 提供原生窗口、文件系统、网络等 API。
- **构建**：`cargo tauri build` 生成跨平台可执行文件。

## 项目结构（关键目录）
```
├─ src/                # Rust 主进程代码
├─ assets/             # 静态资源（图标、默认封面等）
├─ public/             # 前端入口文件（index.html、styles.css、main.js）
├─ docs/               # 项目文档（本目录）
├─ tauri.conf.json    # Tauri 配置文件
└─ Cargo.toml          # Rust 项目依赖
```
