# 使用指南

## 🚀 快速上手

1. **安装依赖**
   ```bash
   pnpm install   # 安装前端依赖
   cargo tauri dev   # 运行开发模式（会自动启动前端 Vite）
   ```
2. **启动应用**
   - 运行 `pnpm tauri dev`，会弹出桌面客户端窗口。
   - 初次启动会自动扫描默认的资源目录（可在设置中修改）。

## 📂 资源管理

- **添加扫描库**：打开设置 → “资源库管理”，选择本地存放 ACGN 资源的文件夹，点击 “扫描”。系统会递归识别视频、漫画、小说等文件并写入 SQLite 数据库。
- **浏览资源**：在左侧侧边栏选择对应的媒体类型，点击条目即可进入播放/阅读页面。

## 🌐 局域网共享

1. 确保所有设备在同一局域网（同一 Wi‑Fi）。
2. 在设置中开启 **局域网服务**，系统会显示本机 IP（如 `192.168.1.23:3000`）。
3. 其他设备打开浏览器，输入该地址即可访问资源库，实现跨设备共享。

## 🎨 UI 特色

- 采用 **玻璃拟态** 设计，配合细腻的微交互（hover、渐变、阴影）。
- 暗黑模式自动适配系统主题，阅读体验更舒适。

## 📋 常见问题 (FAQ)

- **如何手动更新元数据？**
  在资源详情页点击 “编辑元数据”，可手动填写或通过外部 API（MyAnimeList、AniList）同步。
- **资源扫描慢怎么办？**
  大量文件时建议先在设置中限定扫描深度或排除特定文件类型。
- **如何贡献代码？**
  请参考 `docs/开发流程.md`，遵循 Scrum Sprint、分支策略与 CI 检查后提交 PR。
